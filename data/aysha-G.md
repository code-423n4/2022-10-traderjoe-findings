Upgrade Pragma to at Least 0.8.4
Using newer compiler versions and the optimizer gives gas optimizations. Also, additional safety checks are available for free.
==========================================================

>= costs less gas than >
The compiler uses opcodes GT and ISZERO for solidity code that uses >, but only requires LT for >=, which saves 3 gas
